# Kaiten landing

Статический лендинг Kaiten — система управления проектами, задачами и командами.

Собран по дизайн-системе Kaiten V01 (токены, типографика Inter, фиолетовый акцент `#7D4CCF`) и редполитике Kaiten.

## Структура

- `index.html` — весь лендинг (HTML + CSS + минимальный JS для FAQ), без сборки.

## Деплой

Проект подключён к Vercel с автодеплоем из GitHub:

- push в `main` → продакшен-деплой;
- любая другая ветка или PR → preview-деплой.

Локально достаточно открыть `index.html` в браузере.
