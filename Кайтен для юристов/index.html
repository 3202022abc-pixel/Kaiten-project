Более 200 тысяч компаний выбирают Кайтен

Нас часто спрашивают

  • Можно ли использовать Кайтен бесплатно?

    Да, есть бесплатный тариф без ограничения по времени. Можно вести дела, работать командой до 5 человек и создавать неограниченное количество документов. Платные тарифы открывают расширенную аналитику, ИИ-инструменты и установку на собственный сервер. Подробнее о тарифах →

  • Можно ли работать с внешними контрагентами и клиентами?

    Да. Пригласите контрагента с ограниченным доступом — он видит только свой договор, не видит остальные дела и внутренние процессы

  • Как настроить доступы для разных сотрудников?

    Гибко и без ИТ-специалиста. Юристы видят свои дела, ассистенты — только порученные задачи, коллеги из других отделов — только статус своих запросов. За внешних пользователей с доступом только на просмотр платить не нужно

  • Есть ли локальная версия для установки на серверах компании?

    Да. Для организаций с требованиями к хранению данных доступна локальная версия. Все данные остаются на ваших серверах

  • Можно ли перенести данные из таблиц или другой системы?

    Да. Импортируйте из Excel, CSV, Notion, Jira, Trello. Если данных много — команда Кайтена поможет с миграцией

  • Можно ли хранить конфиденциальные документы?

    Да. Кайтен входит в реестр российского ПО, данные хранятся на российских серверах. Доступ к документам настраивается — только нужные люди видят нужные дела
